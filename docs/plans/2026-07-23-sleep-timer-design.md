# 睡眠定时器设计文档

## 定位
Mineradio 底部控制栏新增睡眠定时器功能，支持倒计时和"播完本曲"两种模式，到点渐弱暂停播放，配合视觉粒子渐暗，适合睡前听歌场景。

## 入口
- 底部控制栏 modes 组（右侧辅助按钮区），`immersive-btn` 和 `fullscreen-toggle-btn` 之间
- 月亮图标按钮，激活态亮青色
- 运行中按钮旁显示 mini 倒计时（如 `28:45`）

## 弹出面板（Popover）
与音质/音量面板同风格（玻璃拟态 + 圆角 + 阴影），内容：

### 1. 快速预设
- 15 分 / 30 分 / 45 分 / 60 分 / 90 分 / 自定义
- 胶囊按钮样式，点击直接启动

### 2. 播完本曲
- 独立开关行，与倒计时互斥
- 开启后歌曲自然结束时执行到点动作

### 3. 高级设置（折叠或直接展开）
- 渐弱时长：滑块 0s ~ 30s，默认 8s
- 到点动作：单选「暂停播放」/「停止并清空队列」，默认暂停

### 4. 运行中状态
- 大字号倒计时 `28:45`
- 横向进度条
- 「取消」+「再续 10 分钟」按钮

## 核心逻辑

### 定时器触发流程
```
启动倒计时
  → 每秒 tick 更新 UI
  → 剩余时间 == 渐弱时长 → 开始 rampAudioOutputGain(0, fadeSec*1000)
  → 剩余时间 == 0 → 执行到点动作 + 粒子渐暗
```

### 播完本曲模式
- 监听 audio.onended，如果开启了本曲模式则执行到点动作
- 手动切歌不影响定时器（下一首歌结束依然触发）

### 到点动作
- 暂停：audio.pause() + 保存播放会话（断点续播可用）
- 停止清空：clearQueue() + 停止音频
- 两种动作都伴随粒子 uAlpha 渐暗到 0

### 粒子渐暗
- 渐弱开始时同步 tweenParticleAlpha(当前值, 0, fadeDuration)
- 用户恢复播放时恢复粒子亮度

## 持久化
- localStorage key: `mineradio-sleep-timer-v1`
- 保存内容：lastPreset（上次分钟数）、fadeDuration（渐弱秒数）、endAction（到点动作）、remember（是否记住）
- 运行中状态不持久化（刷新即失效）

## 后续配置集中化
此功能的设置项后续将统一收拢到 `mineradio-settings-v1` 中，作为 sleepTimer 子对象。

## 文件修改清单
- `public/index.html`：HTML 结构 + CSS + JS（单文件架构，全在这一个文件里）

## 不做的事（YAGNI）
- 不做"日落时间自动启动"（依赖天气电台定位，过度设计）
- 不做"播完队列"模式（用例少）
- 不做多个定时器叠加
